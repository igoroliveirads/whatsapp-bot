# WhatsApp Bot - Contributing Guide

Obrigado por considerar contribuir com este projeto!

## Como Contribuir

1. Fork o repositório
2. Crie uma branch para sua feature: `git checkout -b feature/MinhaFeature`
3. Commit suas mudanças: `git commit -am 'Adiciona MinhaFeature'`
4. Push para a branch: `git push origin feature/MinhaFeature`
5. Envie um Pull Request

## Diretrizes

- Mantenha comentários e documentação em Português Brasil
- Mantenha o código em Inglês
- Siga o padrão de código existente
- Adicione testes quando possível
- Atualize a documentação conforme necessário

## Reportando Bugs

- Use a seção de Issues do GitHub
- Descreva o comportamento esperado vs. atual
- Forneça passos para reproduzir o problema
- Inclua screenshots se aplicável

## Sugestões de Melhorias

- Use a seção de Discussions para sugestões
- Explique seu caso de uso
- Mostre exemplos de uso
